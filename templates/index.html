Web-service de previsão do tempo

POST

Realize um post para a rota "/ws" (https://url.com/ws), com as seguintes informações no body. Exemplo:

{ "temperature": "24.85", "humidity": "68.50", "dewpoint": "18.78", "pressure": "1011.44", "speed": "3.99", "direction": "SW", "datetime": "22-Jul-2021 12:54" }

GET

Realize um get para a rota "/ws" (https://url.com/ws), para que possa receber as informações da base em um formato JSON.
